Lista de Definição

Definindo termos

HTML
Hyper Text Markup Language a html, é uma linguagem para marcação de texto e utilizada na construção de páginas web.
CSS
Cascading Style Sheet a css, é um mecanismo utilizado para adicionar estilo a um documento/página web.
Javascript
Javascript o JS, é uma linguagem de proramação usada para deixar as páginas web dinâmicas.
